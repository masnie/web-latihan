Panduan Untuk Git

Panduan ini bertujuan untuk belajar serta membantu memudahkan atau mengingat tentang perintah dasar yang digunakan dalam git

Create

  • -git init
  • -git clone

    • Make a chance

  • -git add
  • -git add .
  • -git commit -m [pesan commit]
  • -git reset [file]
  • -git reset --hard

    • Observe

  • -git status
  • -git diff
  • -git show
  • -git log
  • -git log --oneline

    • Sync

  • -git remote add origin [url repo github]
  • -git remote -v [untuk cek status remote]
  • -git push
  • -git pull origin main [origin [Branch]]
  • -git fetch

    • Branch

  • -git branch [list branch yang ada]
  • -git branch [nama branch] [menambah branch baru]
  • -git checkout [branch] [berpindah branch]
  • -git merge [branch] [menggabungkan branch]
  • -git tag
